New features in exmh since 2.0.3

  • Easier interface for PGP composition
  • Configurable display of dates in the local timezone on messages
  • Support for X-Image-Url headers
  • Middle button on Unseen Window may now do an 'Inc'
  • Subject lines now appear on forwarded messages
  • Better looking icons, in both color and black and white
  • Selectable action for "target" button in folder list
  • May now change autosave interval for Sedit
  • Support for RFC2369
  • Support for newsgroup faces
  • Autoprobe for default paths in installation
  • Disable certain slow graphics operations if the network is slow
  • Use Unicode under Tcl 8.1
  • May now choose precedence of multipart/alternative by changing a resource
  • "New" button in address database browser
  • Added a number of hacks to the misc directory
  • "Clear List..." button in MsgClear
  • Optional automatic aging and deletion of address book entries
  • Optional horizontal scroll bars
  • "Extract Message" command on mouse-right menu for message/rfc822 parts
  • Improvements in MDN handling
  • If Img 1.2 is available, use it for improved inline display of graphics
  • Support access to NNTP servers which require authentication
  • User preference setting for action taken when button 3 ("target button") clicked on a folder
  • Support for LDAP servers
  • Support for wheel mice
